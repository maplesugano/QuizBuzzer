# QuizBuzzer

A quiz buzzer web app built with React and TypeScript. It allows users to join a quiz room, buzz in to answer questions, and keeps track of scores.

## Features
- Create and join quiz rooms
- Buzz in to answer questions (First come, first served)
- Real-time score tracking
- Responsive design for mobile and desktop

## Technologies Used
- React
- TypeScript
- WebSockets for real-time communication
- CSS for styling

## Installation
1. Clone the repository:
```bash
git clone https://github.com/maplesugano/QuizBuzzer.git
```
2. Navigate to the project directory:
```bash
cd QuizBuzzer
```
3. Install dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm start
```
5. Open your browser and navigate to `http://localhost:3000` to access the app.
